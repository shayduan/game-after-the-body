# After the Body
### An Interactive Archive in Seven Voices

> *"The questions were the same for everyone. Not everyone answered."*

## About

**After the Body** is a terminal-style interactive archive game based on an original speculative micro-story. Set in 2056 — three years after the ExoGestation System, a technology that completes pregnancy outside the human body, has entered commercial availability.

The game presents a fictional interview transcript in which seven people of radically different standpoints are asked the same nine questions about what this technology does, who it frees, who it erases, and whether any of it was ever really a choice.

The questions were the same for everyone. Not everyone answered.

The player navigates the archive freely. You choose which questions to explore, whose answers to read, and what to do with the gaps. Some passages have been redacted. Certain words, used by multiple speakers, link across voices to surface contradictions that the transcript cannot contain on its own.


## Running the Interactive Version

Paste `App.jsx` into any React environment:

**Quickest (no setup):**
1. Go to [stackblitz.com](https://stackblitz.com) → Create → React
2. Replace `App.jsx` with the contents of `App.jsx` in this repo
3. Live preview appears instantly — shareable link available

**Local:**
```bash
npx create-react-app after-the-body
cd after-the-body
# replace src/App.js with App.jsx from this repo
npm start
```

**Deploy:**
```bash
npx vercel
```

No external dependencies. Standard React only.

## Context & References

This work sits at the intersection of speculative design, feminist STS, and archival theory. Key references and inspirations:

- Shulamith Firestone, *The Dialectic of Sex* (1970)
- Donna Haraway, *A Cyborg Manifesto* (1985)
- Dunne & Raby, *Speculative Everything* (2013)
- Dublin Core Metadata Initiative — and its limits
- *Her Story* (Sam Barlow, 2015) — as a genre touchstone for archive-as-game
