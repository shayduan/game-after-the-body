# After the Body
### An Interactive Archive in Seven Voices

> *"The questions were the same for everyone. Not everyone answered."*

---

## About

**After the Body** is a terminal-style interactive archive game based on an original speculative micro-story. Set in 2056 — three years after the ExoGestation System, a technology that completes pregnancy outside the human body, has entered commercial availability — the game presents a fictional interview transcript in which seven people of radically different standpoints are asked the same nine questions about what this technology does, who it frees, who it erases, and whether any of it was ever really a choice.

The questions were the same for everyone. Not everyone answered.

The player navigates the archive freely. You choose which questions to explore, whose answers to read, and what to do with the gaps. Some passages have been redacted. Certain words, used by multiple speakers, link across voices to surface contradictions that the transcript cannot contain on its own.

The full interview text is in [`script.md`](./script.md).

## Running the Interactive Version

**Quickest (no setup):**
1. Drop the repo into [StackBlitz](https://stackblitz.com)
2. It detects Vite automatically and gives you a live URL.

**Local:**
```bash
npm install
npm run dev
```
Opens at `http://localhost:5173`

**Deploy (Vercel):**
```bash
npx vercel
```
Or connect the repo directly to [vercel.com](https://vercel.com) for automatic deploys on push.

## Context & References

This work sits at the intersection of speculative design, feminist STS, and archival theory. Key references and inspirations:

- Shulamith Firestone, *The Dialectic of Sex* (1970)
- Donna Haraway, *A Cyborg Manifesto* (1985)
- Dunne & Raby, *Speculative Everything* (2013)
- Dublin Core Metadata Initiative — and its limits
- *Her Story* (Sam Barlow, 2015) — as a genre touchstone for archive-as-game
